# AI Newsletter Generator 📰

Generate complete, structured newsletters
from a single topic using Google Gemini.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- 7 newsletter types (Tech, Startup, Personal
  Brand, Product Launch, Educational, Community,
  College Club)
- 5 tone options
- Customizable section count and length
- Editable subject line, preview text and body
- Download as .txt
- History with full content saved
- Built-in newsletter best practices guide

## Tools Used
- Python, Streamlit, Google Gemini API

## How to Run Locally
pip install streamlit google-generativeai
streamlit run app.py
